# Reference Voice

Place a `reference_voice.wav` file here (3-10 seconds of clear speech).

XTTS v2 will clone this voice for all narration.

## How to create one

Option 1: Record yourself reading a sentence clearly
Option 2: Use any clear voice sample from the internet

Requirements:
- WAV format
- 3-10 seconds long
- Clear speech, minimal background noise
- 22050 Hz or 24000 Hz sample rate preferred
